## Клонирование репозитория

`git clone git@github.com:sakhnov/sf-m3.git` - клонировать удаленный репозиторий в одноименную папку

`git clone git@github.com:sakhnov/sf-m3.git foldername` - клонировать удаленный репозиторий в папку «foldername»

`git clone git@github.com:sakhnov/sf-m3.git .` - клонировать репозиторий в текущую папку (точка в конце)
